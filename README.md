# Ralbum

Generate simple html photo albums.

## Installation

$ gem install ralbum

## Usage

```
Usage: $ ralbum [options]

Default values:
        verbose = false
        title = Photo Album

Options:
    -i, --in-dir [IN_DIR]
    -o, --out-dir [OUT_DIR]
    -v, --[no-]verbose
    -t, --title [TITLE]
        --help
```

## Why?

This was a single-file script that I wrote a while ago and then
forgot about. I needed to use it recently, so I thought I'd
gemify and publish it.
